核心特性

经典 Pascal，现代灵魂

在保留 Pascal 清晰语法的基础上，注入现代语言的强大特性。从类型推断到泛型，从模式匹配到异步编程，Kylix 让 Pascal 焕发新生。

🎯

类型推断

编译器自动推导变量类型，告别冗长的类型声明，让代码更简洁优雅。

var x := 42;
λ

Lambda 表达式

简洁的匿名函数语法，支持闭包捕获和高阶函数，函数式编程触手可及。

(x: Integer) -> x * x
🧬

泛型

类型参数声明，编写类型安全的通用代码。支持泛型类和泛型函数。

TList<T> / TPair<K, V>
🔀

模式匹配

强大的 match 表达式，支持多值匹配、通配符和 when 条件守卫。

match x { 0 => 'zero', _ => 'other' }
⚡

Async / Await

基于 Goroutine + Channel 的异步模型，享受 Go 级并发性能。

async function FetchData(): String
🌐

Web 框架

内置 HTTP 服务器、路由、中间件、JSON 处理，开箱即用。

app.get('/api/users', handler)
🗂️

Map 类型

原生 map[K]V 字典类型，自动初始化，高效键值存储。

var m: map[String]Integer;
🔱

变体类型

Discriminated Union — 类型安全的联合类型，编译为 Go interface + struct。

variant IntLit: Integer; StrLit: String; end
🏛️

类与接口

完整的面向对象支持：继承、多态、属性、构造/析构函数。

class Dog inherits Animal
📝

字符串插值

现代字符串插值语法，表达式直接嵌入字符串。

$"Hello, ${name}!"
🛡️

异常处理

try/except/finally + ON 子句类型化异常捕获。

on E: Exception do WriteLn(E.Message)
🔄

多返回值

函数支持元组返回，解构赋值一步到位。

function Div(a,b: Real): (Real, Boolean)

转译对比

Pascal → Go，一目了然

Kylix 将 Pascal 源码编译为干净、可读、可直接运行的 Go 代码。没有黑魔法，没有运行时依赖。

Pascal 源码 (.klx) 
program WebApp;
uses web;

var app: TServer;
begin
  app := web.createServer(8080);

  app.get('/hello',
    procedure(req: TRequest; res: TResponse)
    begin
      res.json(record
        message := 'Hello, Kylix!';
      end);
    end);

  app.listen();
end.
生成的 Go 代码 (.go) 
package main

import "kylix/stdlib"

var app *stdlib.TServer

func main() {
  app = stdlib.CreateServer(8080)

  app.Get("/hello",
    func(req *stdlib.TRequest, res *stdlib.TResponse) {
      res.Json(struct {
        Message string
      }{Message: "Hello, Kylix!"})
    })

  app.Listen()
}

快速开始

3 步上手 Kylix

一分钟内从安装到运行你的第一个 Kylix 程序。

1

克隆 & 构建

从 GitHub 克隆仓库，一条命令构建编译器。

2

创建项目

使用 kylix new 创建项目模板，自动生成配置和入口文件。

3

编译运行

kylix run 一键编译并运行，即刻看到结果。

# 1. 克隆 & 构建
git clone https://github.com/astra-zhao/kylix.git
cd kylix && go build -o kylix cmd/kylix/main.go

# 2. 创建项目
./kylix new myapp && cd myapp

# 3. 运行
./kylix run
# → Hello, Kylix World!

CLI 工具链

9 个命令，覆盖全流程

从创建项目到编译运行，再到格式化、REPL 交互、LSP 智能服务——完整的开发体验。

# 创建新项目
kylix new myapp

# 编译 & 运行
kylix build              # 编译为 Go 代码
kylix run                # 编译 + 运行
kylix run --keep         # 编译 + 运行（保留 .go 文件）

# 开发辅助
kylix check              # 语法检查（不生成代码）
kylix fmt                # 代码格式化
kylix fmt -w file.klx    # 原地格式化

# 交互 & 编辑器
kylix repl               # 交互式 REPL（支持历史记录）
kylix lsp                # LSP 语言服务器

# 信息
kylix version            # 显示版本
kylix help               # 帮助信息
