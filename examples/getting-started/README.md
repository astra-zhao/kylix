# Getting Started Examples

5 tested Kylix programs from simple to advanced.

## Examples

1. **example1_hello.klx** — Hello World
2. **example2_variables.klx** — Variables and types
3. **example3_functions.klx** — Functions and procedures
4. **example4_loops.klx** — While loops (nested)
5. **example5_class.klx** — Classes and objects

## Run

```bash
kylix build example1_hello.klx
go run example1_hello.go
```
