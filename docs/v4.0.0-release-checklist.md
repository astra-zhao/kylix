## 🐛 Known Issues to Document

List issues that WON'T be fixed in v4.0.0 but should be documented:

1. **LLVM backend - Lambda/closures not supported** (example15_lambda fails)
   - Workaround: Use named functions
   - ETA: v4.1.0 M4

2. **LLVM backend - Tuple LHS assignment stubbed** (multi-return)
   - Workaround: Use record return types
   - ETA: v4.1.0 M4

3. **LLVM backend - `inherited` keyword not implemented**
   - Workaround: Explicit parent method calls
   - ETA: v4.1.0 M4

4. **LLVM backend - No optimization passes** (slower than Go)
   - Workaround: Use Go backend for development
   - ETA: v4.1.0 M4 (`--llvm-opt` flag)
